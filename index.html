Disseny de la base de dades

Aquesta secció descriu el procés de disseny de la base de dades, dividit en tres etapes:

Disseny Conceptual: Modela les entitats i relacions del sistema.

Disseny Lògic-Relacional: Converteix el model conceptual en taules i relacions per a bases de dades relacionals.

Disseny Físic: Inclou el script DDL en SQL per crear la base de dades i les instruccions d'inserció per a proves.

UML. Diagrama de classes

Presenta el diagrama de classes UML, que mostra l'estructura del sistema a través de les seves classes, atributs i relacions, facilitant la comprensió del disseny del software.
